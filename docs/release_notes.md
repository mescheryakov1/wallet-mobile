# Release notes

## Возврат зависимостей к стабильной базе
- Восстановлен `pubspec.lock` с параметрами SDK Dart >= 3.9 / Flutter >= 3.35, чтобы совпадать с требованиями `pubspec.yaml` и исключить даунгрейды зависимостей, появившиеся после пересборки на Flutter 3.24/Dart 3.5.
- Исключены транзитивные даунгрейды ядра (`async`, `boolean_selector` и др.), которые могли ломать сборки и автотесты на актуальной версии SDK.

## Диагностика и воспроизведение
- Задокументирован минимальный сценарий воспроизведения для Android и Windows (`docs/regression_repro.md`), который фиксирует конфликт lock-файла с требуемой версией SDK.

## Конфигурационные изменения
- Lock-файл снова синхронизирован с заявленной версией SDK; дополнительных параметров сборки или платформенных флагов не добавлялось.
